# CatMemo

데스크탑에 고양이가 앉아있는 메모 앱. Tauri 2.x + Vanilla JS로 만들었습니다.

## 기능

- **고양이 위젯** — 화면 어디든 드래그해서 배치, 눈이 커서를 따라다님
- **메모 관리** — 메모 생성/수정/삭제/보관, 색상 커스터마이징, 고정
- **암호화** — Cat Cipher(냥냐냐앙 스타일) 로 메모 내용 암호화/복호화
- **품종 설정** — 고양이 색상을 여러 품종 중 선택 가능
- **멀티 메모** — 여러 메모 창을 동시에 열어둘 수 있음

## 사용법

우클릭으로 컨텍스트 메뉴 열기 → 새 메모 / 메모 목록 / 설정

## 개발

```bash
npm install
cargo tauri dev
```

## 빌드

```bash
cargo tauri build
```
